gitignore   忽略指定文件

配置用户名和邮箱：
    git config --global user.name 'xxxx'    配置用户名
    git config --global user.email 'xxxxx'  配置邮箱
    git config --global --list              查看用户名与邮箱

通用流程：
    git init            初始化

    git add .           提交至暂存区

    git commit -m 说明  将暂存区代码提交到GitHub当中去

    git remote add origin https://github.com/XXXX

    git pull origin master

    git push origin master

操作记录：
    git status          查看当前项目状态
    git log             查看提交记录
            --anthor='name'     查看单个人提交记录

修改文件：
    git add 具体修改的文件名
    (git status  查看状态)
    git commit -m 修改文件
    (it log   查看提交记录)

删除文件：
    git rm 文件名

重命名文件：
    git mv 旧文件名 新文件名

移动文件
    git mv 文件名 文件夹

移动并修改文件名
    git mv 旧文件名 文件夹/新文件名

查看文件信息：
    git log ==pretty=oneline <filename.filetype>    查看具体提交的内容和提交者
    >> id   说明
    git show id     查看

    git log -p <filename.filetype>  查看某个文件修改的具体内容

不再追踪：
    git reset HEAD <filename.filetype>

一键还原：
    git diff        找到文件不同处
    git checkout -- <filename.filetype>

回到上一版本或者指定版本时：
    git reset --hard HEAD^      回到上一版本

    git reset --hard HEAD^^     回到上两个版本

    git reset --hard id         回到id的版本

将某一文件回到指定版本
    git checkout 版本id -- <filename.filetype>
